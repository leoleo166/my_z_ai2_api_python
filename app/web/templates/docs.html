使用示例

curl 示例

curl -X POST "http://localhost:{{ listen_port }}/v1/chat/completions" \ -H "Content-Type: application/json" \ -H "Authorization: Bearer {{ auth_token }}" \ -d '{ "model": "GLM-4.6", "messages": [ {"role": "user", "content": "你好！"} ] }'

Python 示例

import openai # 设置 API 基础 URL 和密钥 openai.api_base = "http://localhost:{{ listen_port }}/v1" openai.api_key = "{{ auth_token }}" # 创建聊天完成 response = openai.ChatCompletion.create( model="GLM-4.6", messages=[ {"role": "user", "content": "你好！"} ] ) print(response.choices[0].message.content)
